# ⌨️ Essential Shortcuts

## 🖥️ System (Hyprland)
*   `SUPER + Q` - Kitty Terminal
*   `SUPER + E` - File Manager
*   `SUPER + D` - This Documentation
*   `SUPER + .` - Emoji Keyboard
*   `SUPER + V` - Clipboard History

## 📟 Terminal (Kitty/Bash)
*   `Ctrl + Shift + C` - Copy
*   `Ctrl + Shift + V` - Paste
*   `Ctrl + L` - Clear Screen
*   `Ctrl + R` - Search History (fzf)
*   `Ctrl + T` - Search Files (fzf)

## 📁 File Manager (Thunar)
*   `Ctrl + T` - New Tab
*   `Ctrl + H` - Show Hidden Files
*   `Alt + Up` - Go to Parent Folder
